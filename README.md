Welcome to BenBot, a discord bot designed by Ben Wu.

This is a general read-me text outlining components of the project and how to start up.

**FILES**
models/
    contains the models used by Mongoose to send to the server

node_modules/
    files included with Node.JS

routes/
    contains all the requests used to contact the server

ban.js
    code containing the main components to ban functionality

index.js
    overarching code that manages BenBot components

package-lock.json
    included with files

package.json
    mainly used to manage package and dependencies

server.js
    contains code that links client to server
